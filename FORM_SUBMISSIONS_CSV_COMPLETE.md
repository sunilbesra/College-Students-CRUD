### 2. CSV File Format
The CSV file must include these columns:


**For UPDATE/DELETE operations:**

1. Select operation type (create/update/delete)

4. Submit for processing

### 4. Monitor Progress
- View processing status at: `http://localhost:8000/form-submissions`
